Tizen Mobile Web API References

The Web API specifications for the Tizen mobile platform consist of a collection of existing standards, such as W3C and Khronos.

In addition to standards-based APIs, the Tizen wearable platform has a specific set of APIs, which provide access to device platform capabilities, enabling a rich Web application development environment.

The following API references provide descriptions of all the APIs that can be used for creating Tizen mobile Web applications:

  • Related Pages

    This reference provides information about the privileges and required features of the Web applications.

  • Device API Reference

    This reference provides the Tizen Device API specifications.

  • UI Framework Reference

    This reference provides tools, such as UI Components, events, effects, and animations, for Web application development.

  • W3C/HTML5 API Reference

    This reference provides the W3C/HTML5 API specifications.

  • Supplementary API Reference

    This reference provides the supplementary API specifications.
