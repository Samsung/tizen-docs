### setting

<div class="views-row views-row-1 views-row-odd views-row-first">

http://tizen.org/feature/systemsetting
------------------------------------------------------------------

The platform returns true for this key, if the device supports the SystemSetting API.


</div>

<div class="views-row views-row-2 views-row-even">

http://tizen.org/feature/systemsetting.home_screen
--------------------------------------------------------------------------------------------

The platform returns true for this key and the http://tizen.org/feature/systemsetting key, if the device supports a way to change or get the picture on the home screen.


</div>

<div class="views-row views-row-3 views-row-odd">

http://tizen.org/feature/systemsetting.lock_screen
--------------------------------------------------------------------------------------------

The platform returns true for this key and the http://tizen.org/feature/systemsetting key, if the device supports a way to change or get the picture on the lock screen.


</div>

<div class="views-row views-row-4 views-row-even">

http://tizen.org/feature/systemsetting.incoming_call
------------------------------------------------------------------------------------------------

The platform returns true for this key and the http://tizen.org/feature/systemsetting key, if the device supports a way to change or get a ringtone for all incoming calls.


</div>

<div class="views-row views-row-5 views-row-odd views-row-last">

http://tizen.org/feature/systemsetting.notification_email
----------------------------------------------------------------------------------------------------------------

The platform returns true for this key and the http://tizen.org/feature/systemsetting key, if the device supports a way to change or get a tone for email notifications.


</div>
