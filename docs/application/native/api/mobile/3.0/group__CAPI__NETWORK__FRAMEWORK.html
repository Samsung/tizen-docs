Network

Overview

API Description
Bluetooth Provides API to manage Bluetooth device.
Connection Provides functions for managing modem data connections.
Curl A client-side URL transfer library supporting http, https, ftp, file URIs and many more protocols.
DNSSD Provides functions for network service discovery using DNSSD.
IoTCon Provides functions for IoT connectivity.
HTTP Provides functions for communication with server according to HTTP protocol.
MTP Provides functions for support PTP(Picture Transfer Protocol) subset of MTP(Media Transfer Protocol).
NFC Provides very short-range radio communication technology.
SSDP Pprovides functions for network service discovery using SSDP.
Smartcard Provides function to communication to SE.
VPN Service Provides functions for managing VPN.
Wi-Fi Provides functions for managing Wi-Fi and monitoring the state of Wi-Fi.
Wi-Fi Direct Provides functions to manage Wi-Fi Direct.
