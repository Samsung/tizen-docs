UI

Overview

API Description
Cairo Open Source Library for 2D vector graphics drawing.
Clipboard History Manager Provides functionalities of copy and paste for applications.
DALi A cross-platform 3D UI Toolkit for embedded systems.
EFL A set of graphics librarie.
EFL UTIL Provides functions to get the level of given notification windows and to change the notification window levels with the given values.
Efl Extension Provides functionalities to enhance the EFL libraries.
External Output Manager Provides APIs for external outputs.
FontConfig Font-handling library to let applications find a font or a closely matching font.
Freetype Text-rendering library.
HarfBuzz Provides functions for text shaping.
Minicontrol Provides functions to manage minicontrol.
SDL Open Source Library for cross-platform development library. provides low level access to audio, keyboard, mouse, and graphics hardware
TBM Surface Provides user interface for The TBM Surface.
Tizen WS Shell Provides APIs to communicate with the window manager.
UI View Manager This module provides functionalities for applications view management.
Vulkan Vulkan is a new generation graphics and compute API that provides high-efficiency, cross-platform access to modern GPUs used in a wide variety of devices from PCs and consoles to mobile phones and embedded platforms.
