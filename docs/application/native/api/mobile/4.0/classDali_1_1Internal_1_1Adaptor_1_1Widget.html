Detailed Description

This is the internal base class of custom widget.

It will provides several widget instance lifecycle virtual functions which the user can override.

User should override OnCreate function and create scene for custom widget.

Plus, Implements ConnectionTrackerInterface so that signals (typically connected to member functions) will be disconnected automatically when the control is destroyed.

Since:
4.0, DALi version 1.3.5
