Tizen WS Shell
UI

The Tizen WS Shell (tzsh) provides APIs to communicate with the window manager.

Required Header

#include <tzsh.h>

Overview

The Tizen WS Shell API provides functions to create/destroy the tzsh instance to communicate the window manager.
