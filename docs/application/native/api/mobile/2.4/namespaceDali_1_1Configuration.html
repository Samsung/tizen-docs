enum Dali::Configuration::ContextLoss

Enumeration about application context loss policy.

Since :
2.4
Enumerator:
APPLICATION_HANDLES_CONTEXT_LOSS 

Application will tear down and recreate UI on context loss and context regained signals. Dali doesn't need to retain data.

Since :
2.4
APPLICATION_DOES_NOT_HANDLE_CONTEXT_LOSS 

Application expects Dali to retain data ( increased memory footprint )

Since :
2.4
