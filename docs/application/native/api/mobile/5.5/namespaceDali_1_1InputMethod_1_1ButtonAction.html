Detailed Description

Enumeration for specifying what the Input Method "action" button functionality is set to.

The 'Action' button is traditionally the [RETURN] or [DONE] button. Not all these actions are supported by all systems. Setting a custom label will still require one of these actions to be set.

Since:
5.0, DALi version 1.3.20
