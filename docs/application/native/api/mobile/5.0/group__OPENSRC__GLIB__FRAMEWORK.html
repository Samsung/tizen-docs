Glib
Base

Overview

Application building blocks which add data types and other programming facilities for C-language programs.
Version: 2.52.2
Reference: https://developer.gnome.org/glib/2.52/

In some cases such as internet access, your application requires some privileges. For more information, see the Non-API Bound Privileges.

Warning:
Some of glib APIs (Gdbus APIs: g_bus_*, g_dbus_*, and g_test_dbus_*) are removed due to security issues.
Please do not use Gdbus APIs in your Tizen applications, and use Tizen Native APIs instead.
Please also refer to these web pages (Application Framework IPCs (Message Port, Data Control, etc.)).
If you cannot find any proper APIs, please post a request on Tizen Community with detailed description about purpose of use.
