_Evas_Object_Box_Data Struct Reference
Box Smart Object

Detailed Description

The structure type that augments clipped smart object's instance data, providing extra members required by generic box implementation. If a subclass inherits from Evas_Object_Box_Api, then it may augment Evas_Object_Box_Data to fit its own needs.
