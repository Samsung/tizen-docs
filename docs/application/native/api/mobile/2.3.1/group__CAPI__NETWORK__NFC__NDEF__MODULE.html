NDEF
NFC

The NDEF api provides functions to make NDEF record and NDEF message.

Required Header

#include <nfc.h>

Overview

The NDEF api provides functions to make NDEF record and NDEF message.

  • create NDEF record and retrieve specific field of NDEF record
  • create NDEF message with NDEF records
APIs Description
Record Record API provides NDEF record and retrieve specific field of NDEF record
Message Message API provides functions to create NDEF message and insert/remove records in NDEF message
