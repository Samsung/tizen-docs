Ecore Buffer Queue functions
Ecore_Buffer - Graphics buffer functions

Deprecated:
Deprecated since tizen 6.0. Ecore Buffer Queue is a queue which connects processes for sharing Ecore_Buffer. One process (related object is Ecore_Buffer_Provider) has rear terminal position of Ecore_Buffer Queue which can enqueue the Ecore_Buffer, and the other process (related object is Ecore_Buffer_Consumer) has front terminal position of Ecore_Buffer_Queue which can dequeue the Ecore_Buffer.
