What is Elementary?

This is a VERY SIMPLE toolkit. It is not meant for writing extensive desktop applications (yet), only small simple ones with simple needs.

It is meant to make the programmers work almost without much brain usage but give them lots of flexibility.

  • Getting Started - Go here to quickly get started with writing apps.

Organization

One can divide Elementary into three main groups:

  • Infrastructure list - These are modules that deal with Elementary as a whole.
  • Widget list - These are the widgets you compose your UI out of.
  • Containers List - These are the containers which hold the widgets.
