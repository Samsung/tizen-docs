Efl Extension
UI

This module provides functionalities to enhance the EFL libraries.

Required Header

#include <efl_extension.h>

Overview

EFL Extension provides functionalities to enhance the EFL libraries. It includes device-specific features (like support for hard back key) or profile-specific features (circular UI for wearable device).
