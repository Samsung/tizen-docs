#include <device/led.h>
#include <device/power.h> #include <device/ir.h>

Overview

The DEVICE API provides functions to control devices or to get status of devices.

This API allows checking of the following parameters:

  • Battery
  • Display
  • Haptic
  • LED
  • Power
  • IR
