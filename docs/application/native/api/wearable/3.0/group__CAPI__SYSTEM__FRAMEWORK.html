System

Overview

API Description
Device Provides functions to control devices or to get status of devices.
Feedback Provides functions to play sound or vibration associated with properties.
Media key Provides functions for getting the status of the media key when the key is changed.
Runtime information Provides functions to obtain runtime information.
Sensor Provides functions to start/stop sensors and receive sensor information.
Storage Provides functions to get storage information.
System Information Provides functions which can be used to obtain device information.
System Settings Provides APIs for sharing configuration over a system.
T-trace Provides functions for writing trace message to the system trace buffer.
dlog Provides functions for sending log output.

For more information on the System submodule features, see System Guide.
