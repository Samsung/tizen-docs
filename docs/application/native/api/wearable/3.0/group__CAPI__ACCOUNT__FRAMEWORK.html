Account

Overview

API Description
Account Manager Provides CRUD (Create, Read, Update, Delete) account management functionality.
FIDO Client Provides APIs for FIDO(Fast Identity Online) UAF(Universial Authentication Framework) Standards.
Sync Manager Provides APIs for managing the sync operations.
libOAuth Provides functions for the OAuth Core RFC 5849 protocol.
OAuth 2.0 Provides functions for the OAuth Core RFC 6749 protocol.

For more information on the Account submodule features and the macros, see Account Framework programming guides and tutorials.
