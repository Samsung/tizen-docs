## SpreadType Enum

Defines the different methods of spreading the colors of a gradient across its area.

```csharp
public enum SpreadType
```
### Fields

<a name='Tizen.UI.Primitives2D.SpreadType.Pad'></a>

`Pad` 0

The gradient colors will be distributed evenly across the area, with any remaining space filled with the color of the first color stop.

<a name='Tizen.UI.Primitives2D.SpreadType.Reflect'></a>

`Reflect` 1

The gradient colors will be reflected at the edge of the area, creating a mirrored pattern.

<a name='Tizen.UI.Primitives2D.SpreadType.Repeat'></a>

`Repeat` 2

The gradient colors will be repeated in a tiled pattern across the area.
