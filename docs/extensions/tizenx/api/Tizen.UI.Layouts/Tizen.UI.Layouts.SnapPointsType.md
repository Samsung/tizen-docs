## SnapPointsType Enum

Specifies the behavior of snap points when scrolling.

```csharp
public enum SnapPointsType
```
### Fields

<a name='Tizen.UI.Layouts.SnapPointsType.Mandatory'></a>

`Mandatory` 1

Indicates that content always snaps to the closest snap point to where scrolling would naturally stop, along the direction of inertia.

<a name='Tizen.UI.Layouts.SnapPointsType.MandatorySingle'></a>

`MandatorySingle` 2

Indicates the same behavior as Mandatory, but only scrolls one item at a time.

<a name='Tizen.UI.Layouts.SnapPointsType.None'></a>

`None` 0

Indicates that scrolling does not snap to items.
