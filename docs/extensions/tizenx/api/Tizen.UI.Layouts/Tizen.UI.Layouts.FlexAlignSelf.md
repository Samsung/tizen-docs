## FlexAlignSelf Enum

Enumerates values that control how and whether a child element overrides alignment attributes applied by its parent.

```csharp
public enum FlexAlignSelf
```
### Fields

<a name='Tizen.UI.Layouts.FlexAlignSelf.Auto'></a>

`Auto` 0

Indicates that the element will use the alignment supplied by the FlexAlignItems value provided to its parent.

<a name='Tizen.UI.Layouts.FlexAlignSelf.Center'></a>

`Center` 2

Indicates that the element will be centered within the parent.

<a name='Tizen.UI.Layouts.FlexAlignSelf.End'></a>

`End` 4

Indicates that the element will be placed against the end of the parent.

<a name='Tizen.UI.Layouts.FlexAlignSelf.Start'></a>

`Start` 3

Indicates that the element will be placed against the start of the parent.

<a name='Tizen.UI.Layouts.FlexAlignSelf.Stretch'></a>

`Stretch` 1

Indicates that the element will stretch from start to end of the parent.
