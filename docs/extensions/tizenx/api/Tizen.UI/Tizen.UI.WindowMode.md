## WindowMode Enum

The WindowMode enum represents the possible modes for a window.

```csharp
public enum WindowMode
```
### Fields

<a name='Tizen.UI.WindowMode.Opaque'></a>

`Opaque` 0

Indicates that the window is opaque, meaning that it displays a solid color and does not allow any other windows to be seen through it.

<a name='Tizen.UI.WindowMode.Transparent'></a>

`Transparent` 1

Indicates that the window is transparent, meaning that it displays a color that is mixed with the color of the window behind it. This allows other windows to be seen through the transparent window.
