## TokenPropertyNames Class

Provides property names used in token-related events to indicate which property has changed.

```csharp
public static class TokenPropertyNames
```

Inheritance [System.Object](https://docs.microsoft.com/en-us/dotnet/api/System.Object 'System.Object') &#129106; TokenPropertyNames

### Remarks
This static class contains predefined property name constants that are used to specify   
which token property has been modified in events such as [TokenPropertyChangedEventArgs](Tizen.UI.TokenPropertyChangedEventArgs.md 'Tizen.UI.TokenPropertyChangedEventArgs').
