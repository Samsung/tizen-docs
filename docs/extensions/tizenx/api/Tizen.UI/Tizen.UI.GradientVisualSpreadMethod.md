## GradientVisualSpreadMethod Enum

Specifies the method used to spread the gradient colors across the area being painted.

```csharp
public enum GradientVisualSpreadMethod
```
### Fields

<a name='Tizen.UI.GradientVisualSpreadMethod.Pad'></a>

`Pad` 0

Uses the terminal colors of the gradient to fill the remainder of the quad.

<a name='Tizen.UI.GradientVisualSpreadMethod.Reflect'></a>

`Reflect` 1

Reflects the gradient pattern start-to-end, end-to-start, start-to-end, etc. until the quad is filled.

<a name='Tizen.UI.GradientVisualSpreadMethod.Repeat'></a>

`Repeat` 2

Repeats the gradient pattern start-to-end, start-to-end, start-to-end, etc. until the quad is filled.
