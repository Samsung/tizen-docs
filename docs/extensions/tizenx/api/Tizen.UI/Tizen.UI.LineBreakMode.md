`Character` 1

Character-wrapping mode. Lines are broken at character boundaries.

<a name='Tizen.UI.LineBreakMode.Hyphenation'></a>

`Hyphenation` 2

Hyphenation mode. Words that cannot fit on a line are broken by inserting hyphens.

<a name='Tizen.UI.LineBreakMode.Mixed'></a>

`Mixed` 3

Mixed mode. A combination of word-wrapping and character-wrapping.

<a name='Tizen.UI.LineBreakMode.Word'></a>

`Word` 0

Word-wrapping mode. Lines are broken at word boundaries.
