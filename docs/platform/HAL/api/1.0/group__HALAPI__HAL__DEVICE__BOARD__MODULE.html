Board
Device

The Board provides functions for controlling and getting information about board.

Required Header

#include <hal-device-board.h>

Overview

The Board provides the way to get device board information.

The Board functions provide board information including:

  • Load/Unload for board hal-backend module
  • Getter for device serial number
  • Getter for device revision
  • Setter for BOOT_SUCCESS flag
  • Getter for boot mode
  • Getter for boot reason
  • Getter for current partition
  • Setter for switch partition
  • Getter/Setter for a/b partition cloned flag
  • Getter/Setter for partition status
  • Getter/Setter for upgrade progress status
  • Getter/Setter for upgrade state
  • Getter/Setter for upgrade type

For more information on the Board features and the macros, see HAL Board programming guides and tutorials.
