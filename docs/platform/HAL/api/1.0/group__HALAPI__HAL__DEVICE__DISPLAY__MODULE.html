Display
Device

The Display provides functions for getting display information.

Required Header

#include <hal-device-display.h>

Overview

The Display provides getter/setter properties related to display. It also supports the API for an application to receive the display event by callback function from the system.

The Display functions provide display information including:

  • Load/Unload for display hal-backend module
  • Getter/Setter for brightness
  • Getter/Setter for display state
  • Getter/Setter for display panel mode
  • Getter/Setter for display image effect
  • Getter/Setter for display frame rate
  • Getter/Setter for display white balancing
  • Getter/Setter for display rotation angle
  • Getter for max brightness
  • Getter for min/max frame rate
  • Getter for aod brightness
  • Getter for aod mode
  • Getter for auto brightness
  • Setter for multi brightness

For more information on the Display features and the macros, see HAL Display programming guides and tutorials.
