System

The DEVICE module provides functions to control devices or to get status of devices from the hal-backend.

Overview

Module Description
Battery Provides functions to receive battery events or battery information
Bezel Provides functions to control and get bezel related
It is used from the wearable
Board Provides functions to control device board settings or get information boot related
Display Provides functions to control display
External Connection Provides functions to control extcon devices
Haptic Provides functions to control vibration functionality of a device
Input Provides functions to control input devices
Ir Provides functions to control a IR transmitter
Led Provides functions to control the attached led device
Memory Provides functions to get memory information
Power Provides functions to get power information
Thermal Provides functions to get information about the temperature
Touchscreen Provides functions to control functionality related touchscreen
