Type Module Description
Tizen Package Common Error This package provides Tizen error codes and some functions related with error information.

Open Source alsa-lib The Advanced Linux Sound Architecture (alsa-lib) is a library to interface with ALSA in the Linux kernel and virtual devices using a plugin system.
Open Source coregl This package provides EGL/OpenGL ES debugging and Performance logging
Open Source libdlog This package adds logs in Tizen platform.
Open Source libdrm The libdrm provides a userspace library for accessing the direct rendering manager (DRM) on operating systems that support the ioctl interface.
Open Source libsystemd This package provides only the libudev. It helps to access device node, monitoring/filtering uevent and lookup devices and its properties.
Open Source libtbm The library for Tizen Buffer Manager.
Open Source libtpl-egl Tizen Porting Layer (a.k.a TPL) is a linkage between the underlying window system and the EGL porting layer or Vulkan WSI layer.
Open Source libusb The libusb is a C library that provides generic access to USB devices.
